Summary:

This week we learned CSS. We learned about changing text types, colors, adding backgrounds, and how to add a link in HTML code that links to CSS. In regards to CSS colors, the book covered different color values, contrast, and opacity. We learned about foreground colors, background colors, and how to change text colors. The book also covers CSS selectors-- Child, ID, Class,and Type are some examples. We also learned about changing the font family, size, and other ways to manipulate the text on a page.
