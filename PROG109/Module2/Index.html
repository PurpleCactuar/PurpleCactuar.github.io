Module 2 - Introduction to Web Development

Summary:

For the second week of class, we covered chapters 6 through 8. This week was about forms, tables, META tag, DocType, and iFrame. The main takeaway for the META tag is that it defines the data about a HTML document. The metadata is information used by browsers, search engines, and other web services. iframe is an inline used to embed another document within the current HTML document. DOCTYPE is a declaration to the browser so that the browser knows what document to expect. Forms are used to collect user data/input and allows them to interact with your page. Tables are used to organize data.
